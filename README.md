# BlogSystem
## 博客文章系统
#### ASP.NET Framework (C#) + Web + Layui + MySQL

### 使用说明:
1.先创建一个MySQL数据库, 名为mydayebase,然后使用数据脚本mydayebase.sql生成数据表和数据 \
2.在Web.config修改连接字符串对应你本地数据库即可运行.
### 补充信息:
①本项目使用MySql数据库, 脚本文件为mydayebase.sql. \
②默认管理员为: \
用户名:admin \
密码:admin
